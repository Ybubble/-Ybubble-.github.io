Github+hexo搭建博客

本文记录了基于Github和hexo搭建博客的过程，以及在该过程中遇到的问题和解决方法

工具准备

  • Git：官网下载页面-https://git-scm.com/downloads
  • NodeJS：官网下载页面-https://git-scm.com/downloads
  • Github账号：账号申请-https://github.com/

Github准备

点击主页右上角的头像，点击“Your repositories”

点击“new”，填写信息新建仓库（注意仓库名需与用户名一致），然后点击“Create repository”

Fluid主题

安装：fluid官网有详细配置步骤：fluid-dev/hexo-theme-fluid: :ocean: 一款 Material Design 风格的 Hexo 主题 / An elegant Material-Design theme for Hexo (github.com)，本文采用方法二

个性化设置：

  1. 浏览器标题

    修改根目录下 _config.yml 中的 title 字段

  2. 博客标题

    修改主题目录 themes\fluid 下 _config.yml 中的 blog_title 、text字段

注意

更换主题

若更换主题后多次尝试但并未生效，可以尝试通过 hexo clean 来清除缓存文件 (db.json) 和已生成的静态文件 (public)
